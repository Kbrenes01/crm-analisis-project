Herramienta: CRM Predictivo de Ventas en R

Esta herramienta busca automatizar el análisis de ventas y clientes que actualmente se realiza manualmente en hojas de Excel.
Por ejemplo, los reportes de ventas se elaboran de forma manual sumando datos en distintas hojas, lo que consume tiempo y puede causar errores.

Con esta aplicación en R, se integran automáticamente las tablas de clientes y ventas, se generan KPIs (ventas totales, promedio por cliente, clientes activos), gráficos mensuales y un modelo predictivo simple que proyecta las ventas futuras de forma rápida y confiable.

Funciones personalizadas (KPI y utilidades)

En esta parte, desarrollamos funciones personalizadas para obtener indicadores de negocio. Creamos una función que calcula los KPI más importantes, como las ventas totales, el número de clientes activos y la venta promedio por cliente. Además, generamos una tabla con los clientes que más compran, lo que ayuda a identificar a los más importantes para la empresa. Estas funciones permiten automatizar el análisis sin tener que hacerlo manualmente en Excel.

# KPI: ventas totales, ventas promedio por cliente, clientes activos

calc_kpis <- function(df_crm) {
  total_ventas <- sum(df_crm$`Monto Total`, na.rm = TRUE)
  clientes_unicos <- n_distinct(df_crm$`ID Cliente`)
  ventas_promedio_cliente <- df_crm %>%
    group_by(`ID Cliente`) %>%
    summarise(total = sum(`Monto Total`, na.rm = TRUE)) %>%
    summarise(prom = mean(total, na.rm = TRUE)) %>%
    pull(prom)
  list(
    total_ventas = total_ventas,
    clientes_unicos = clientes_unicos,
    ventas_promedio_cliente = ventas_promedio_cliente
  )
}

# Top N clientes por ventas

top_clients <- function(df_crm, n = 10) {
  df_crm %>%
    group_by(`ID Cliente`) %>%
    summarise(Cliente = first(`Nombre del Cliente`),
              Ventas = sum(`Monto Total`, na.rm = TRUE)) %>%
    arrange(desc(Ventas)) %>%
    slice_head(n = n)
}

# Función para crear tabla mensual por cliente 
ventas_mensual_por_cliente <- function(df_crm) {
  df_crm %>%
    group_by(`ID Cliente`, AñoMes = floor_date(Fecha, "month")) %>%
    summarise(Ventas = sum(`Monto Total`, na.rm = TRUE)) %>%
    ungroup()
}

Análisis predictivo

Análisis predictivo simple con lm() (modelo lineal) para encontrar relaciones entre variables y predecir valores futuros.

En esta parte usamos un modelo lineal simple para predecir las ventas futuras.Primero, ordenamos los meses en una secuencia numérica para que R entienda la progresión del tiempo. Luego, aplicamos la función lm() para encontrar la tendencia general de las ventas. Finalmente, generamos predicciones para los próximos tres meses, mostrando no solo la estimación, sino también un rango posible con un 95% de confianza.

# Preparar datos para lm: convertir AñoMes a índice numérico (t)

ventas_mensual <- ventas_mensual %>%
  mutate(t = seq_len(n()))  # 1,2,3,...

# Ajustar modelo lineal: Ventas ~ t

modelo_lm <- lm(Ventas ~ t, data = ventas_mensual)
summary(modelo_lm)  # ver R^2, coeficientes

# Predecir próximos 3 meses
h <- 3
ultimo_t <- max(ventas_mensual$t)
newdata <- data.frame(t = (ultimo_t + 1):(ultimo_t + h))
predicciones <- predict(modelo_lm, newdata = newdata, interval = "prediction", level = 0.95)
pred_df <- data.frame(
  AñoMes = seq(max(ventas_mensual$AñoMes) %m+% months(1), by = "month", length.out = h),
  pred = predicciones[, "fit"],
  lwr = predicciones[, "lwr"],
  upr = predicciones[, "upr"]
) # Usa el modelo para estimar las ventas futuras, incluyendo un intervalo de confianza del 95% (valores mínimo y máximo esperados).
